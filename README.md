Hospital Appointment System
->This is a hospital appointment system. It allows patients to book appointments with doctors.

Installation
->To install, 
->run npm install.

Usage
->To run the server, 
->run npm start.

Testing
->To run the tests,
-> run npm test.

Contributing
->To contribute,
